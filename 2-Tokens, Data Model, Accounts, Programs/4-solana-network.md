### Solana Network:

**Three types:**

    1. MainNet --> Main Solana Network. (Real One)
    2. DevNet --> Same as MainNet. But only for testing purpose.
    3. TestNet --> Live production environment


**Good Read:**
***[Clusters and Public RPC Endpoints]***

Link: https://solana.com/docs/references/clusters

### MainNet:
```
|User|  -->  |MainNet RPC URL|  <-->  |Real Blockchain Miners|
```

### DevNet:
```
|User|  -->  |DevNet RPC URL|  <-->  |DevNet Blockchain Miners|
```

### TestNet:
```
|User|  -->  |TestNet RPC URL|  <-->  |Lcoal-Machine Test-Validator|
```
